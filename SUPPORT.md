# Support

## Getting Help
- **Bug reports** → Open a GitHub Issue using the Bug Report template.
- **Feature requests** → Open a GitHub Issue using the Feature Request template.
- **Questions** → Open a GitHub Discussion in the Q&A category.
- **Security issues** → See `SECURITY.md`. Do NOT open a public issue.

## Before Opening an Issue
1. Search existing issues and discussions.
2. Check `docs/ai/mistakes.md` for known issues.
3. Check the repository's `README.md`.
