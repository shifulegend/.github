# Security Policy

## Reporting a Vulnerability
**Do NOT open a public issue for security vulnerabilities.**

Please report security issues by:
1. Opening a [GitHub Security Advisory](https://github.com/shifulegend/REPO_NAME/security/advisories/new) in the affected repository.
2. Including: description, reproduction steps, potential impact, and suggested fix if known.

## Response Time
- Acknowledgement within 48 hours.
- Assessment and fix timeline communicated within 7 days.

## Scope
- Vulnerabilities in application code
- Dependency vulnerabilities
- Secrets accidentally committed

## Out of Scope
- Issues in dependencies not under this project's control (report upstream)
- Issues requiring physical access
